Descripción de las Tecnologías Utilizadas

En la creación de esta página se han utilizado diversas tecnologías para lograr una experiencia interactiva y moderna. HTML (HyperText Markup Language) es el lenguaje utilizado para estructurar el contenido de la página web. Es la base fundamental de cualquier sitio web, permitiendo definir encabezados, párrafos, enlaces y otros elementos.

CSS (Cascading Style Sheets) se emplea para diseñar y dar estilo a la página. Permite controlar colores, fuentes, márgenes y disposición de los elementos, mejorando la estética y la accesibilidad de la página.

JavaScript es el lenguaje de programación utilizado para agregar interactividad a la página. Con JS se realizan dinámicas como la carga de contenido sin recargar la página, interacciones con el usuario, y validación de formularios.

Leaflet.js es una librería de JavaScript utilizada para mostrar mapas interactivos. En este proyecto, se emplea para visualizar el mapa del municipio de Chipaque y agregar un polígono de interés sobre él. Leaflet es ligera y fácil de usar, permitiendo crear mapas dinámicos con poco código.
